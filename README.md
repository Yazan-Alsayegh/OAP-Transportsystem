## Funksjoner

- Brukerautentiseringssystem
- Kundeadministrasjon
- Ansattadministrasjon
- Ordrebehandling

- Kontorstyring

## Teknologier Brukt

- Java 8
- Swing for UI-komponenter
- FlatLaf for moderne UI-tema
- MySQL for databaselagring
- Apache POI for Excel-integrasjon
- Log4j for logging
